# Project Page

This directory is a self-contained static site intended for GitHub Pages deployment.

## Publish

1. Push the repository to GitHub.
2. In repository settings, open `Pages`.
3. Set the source to deploy from the branch you want and choose the `/docs` folder.
4. Save; GitHub Pages will publish `docs/index.html`.

## Replace Later

- `docs/index.html`: author list, link buttons, citation block, and video placeholders.
- `docs/assets/`: current teaser/method/results images copied from the repository for a working first version.
